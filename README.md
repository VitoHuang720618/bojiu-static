# Bojiu Static - 靜態資源與代理服務架構

本專案為 Bojiu 靜態站點，採用 Vite + Vue 3 進行開發，並透過 Nginx 進行部署與 API 代理。

## 1. Nginx 服務配置 (Server Configuration)

服務網址：`bojiu.vito.website`

### 核心路徑規劃 (Path Mapping)
* **根目錄 (Root)**: `/var/www/bojiu-static/dist` (指向 Vite 建置後的靜態檔案)。
* **靜態上傳資源 (Uploads)**: `/var/www/bojiu-static/uploads/` (透過 `alias` 映射，支援軟連結 `disable_symlinks off`)。
* **API 代理 (Proxy)**: 將前端 `/api/hostnames` 轉發至遠端負載平衡節點。

### 關鍵技術實作
1. **CORS 跨域處理**: 
   * 自動隱藏後端 `Access-Control-Allow-Origin`。
   * 動態映射 `Origin` 並開啟 `Credentials` 支援，確保 Cookie 正常運作。
2. **SNI 補強 (SNI Support)**:
   * 開啟 `proxy_ssl_server_name on`。
   * 強制設定 `proxy_ssl_name` 為目標 Domain，解決多數 502 握手失敗問題。
3. **身分偽裝 (Identity Spoofing)**:
   * 轉發正確的 `Host` 與 `Referer`。
   * 完整傳遞 `Set-Cookie` 與 `User-Agent`。
4. **Vue Router 支援**:
   * 配置 `try_files $uri $uri/ /index.html`，支援 History 模式。
